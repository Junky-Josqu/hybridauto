# Definition-Wortherkunft

## Definition
 > Ein **Hybridelektrokraftfahrzeug**(Hybridauto) ist ein Fahrzeug, dass mindestens von einem **Elektromotor** und einem **anderen Motor** angetrieben wird.

## Wortherkunft
  Hybrid = hybrida(lat.) = Mischling
